What is a Side Effect?

A React component is meant to be a pure function.

A pure function:

Takes input (props/state)

Returns JSX

Does not affect anything outside itself

But when a component does something like:

Making an API call

Updating localStorage

Accessing or modifying the DOM

Setting timers (setTimeout, setInterval)

Adding event listeners

üëâ This is called a Side Effect.


How useEffect Works

useEffect takes two arguments:

A function ‚Üí contains side effect logic

A dependency array ‚Üí controls when it runs

Different Behaviors:
Dependency Array	Behavior
[]	Runs only once (on mount)
[value]	Runs when value changes
No array	Runs after every render
